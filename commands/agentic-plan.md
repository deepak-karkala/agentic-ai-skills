  autonomy boundaries, orchestration pattern, and tool design. Routes to
  the agentic-system-design skill with the agent-systems-architect subagent
  available for deeper analysis.
---

Invoke the agentic-ai-engineering:agentic-system-design skill.

The user wants to design a new agentic system. Begin by understanding:
1. What task or workflow the agent will automate
2. The acceptable autonomy level and reversibility of actions
3. Any existing infrastructure, tools, or APIs the agent must use
4. Scale, latency, and cost constraints

Then run the agentic-system-design skill workflow, using the agent-systems-architect subagent for architecture decomposition if the design is non-trivial.

Produce a structured architecture plan. If the design warrants it, generate the architecture-review HTML artifact.
